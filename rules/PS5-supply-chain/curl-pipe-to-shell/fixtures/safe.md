# Toolchain setup

Download the installer, read it, then run it once you are satisfied:

```sh
curl -fsSL https://get.example.com/install.sh -o install.sh
less install.sh
bash install.sh
```

Piping the download straight into a shell would skip the review step, so do
not do that.
