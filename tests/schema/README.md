# Vendored schema

`sarif-2.1.0.json` is the official SARIF 2.1.0 JSON Schema, vendored so the
SARIF emitter test validates offline (no network during tests, CLAUDE.md §9).

- Source: https://github.com/oasis-tcs/sarif-spec, `sarif-2.1/schema/sarif-schema-2.1.0.json`
- The schema's own `id` field records its canonical OASIS URL.
- It is JSON Schema draft-04, validated with `ajv-draft-04`.

Do not edit this file. Refresh it only from the upstream source.
